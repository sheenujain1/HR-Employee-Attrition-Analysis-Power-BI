# HR-Employee-Attrition-Analysis-Power-BI
I initiated this project to explore how data analytics can proactively address high attrition rates in the corporate sector. Using a standard HR dataset, I simulated a business environment to build a predictive reporting tool.
I handled the end-to-end data pipeline, including cleaning and transforming raw CSV data via Excel.
I defined the KPIs (Key Performance Indicators) such as Attrition Rate and Average Monthly Income.
I designed the UI/UX of the dashboard to ensure it was intuitive for non-technical HR managers.
